# PFE 3D Viewer

## Dependencies

- `cmake`
- [`glbinding` (`v3.1.0`)](https://github.com/cginternals/glbinding/releases/tag/v3.1.0)

## Usage

### Install dependencies

- **On macOS:**
	- If needed, install `brew` _([steps](https://docs.brew.sh/Installation))_
	- Run the command:
		```
		brew install cmake glbinding
		```

### Compilation

- **On Linux/macOS:**
	- Run the command:
		```
		./compile.sh
		```
